After tax Calc

Alright here we are with a nice salary tax calculator. The difference from traditional calculators, is that it allows you to check salary tax progression as years pass by.

Somebody could suggest that:
  • React is overkill given the small size of he project
  • and that an equation could be used for the calculation of the salary instead of having a flat file of salaries
But this project was done solely for the pleasure of learning new stuff! I had the opportunity to work with React, React Testing Library and Puppeteer!

The intention is to host several EU countries so somebody can compare and contrast salaries across EU.

The source code is available in Github. Star the mofo!
